> ### 문제
>
> 1부터 어떤 양의 정수 n까지의 정수를 모두 곱한 것을 말하며 n!로 나타낸다.
>
> 
>
> 0! = 1
>
> 1! = 1
>
> 2! = 2
>
> n! = n * (n-1)!
>
> ：
>
> 와 같이 정의된다.
>
>  
>
> 예로 4! = 4×3×2×1 = 24 이다.
>
>  
>
> n! 이 주어졌을 때 결과를 출력하는 프로그램을 작성하라.
>
>  
>
> \* 결과가 int범위를 넘는 경우 
>
> 자료형 **long long** 
>
> 입력/출력 서식문자 **%lld** 
>
> 를 사용한다.
>
> 
>
> ### 출력형식
>
> n!에 대한 계산 결과를 "출력예"처럼 과정을 출력하고 마지막에 결과를 출력한다.
>
> 
>
> ### 출력 예
>
> ```
> 4! = 4 * 3!
> 3! = 3 * 2!
> 2! = 2 * 1!
> 1! = 1
> 24
> ```

```python
def factorial(N):
    if N == 1:
        print('1! = 1')
        return 1
    else:
        print(f'{N}! = {N} * {N - 1}!')
        return N * factorial(N - 1)


N = int(input())
print(factorial(N))
```

재귀를 너무 못하는 것 같아서 정올 사이트에 있는 기초부터 시작했다.

가장 기초적인 팩토리얼인데, print를 찍어주기만 하면 된다.
