Low Level 언어 중 떠오르는 Rust를 접해보았다.



`$ cargo new calculator`

- `$ cargo run`

- `$ rustc ./src/main.rs` => compiles

  `$ ./main` => main.exe

- `$ cargo build`

- `$ cargo build --release` => for production
